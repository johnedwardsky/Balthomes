# Инструкция по запуску сайта

## 1. Как загрузить сайт на GitHub (чтобы он был в интернете)
1.  Зарегистрируйтесь или войдите на [github.com](https://github.com).
2.  Нажмите кнопку **New repository** (или "+" вверху справа -> New repository).
3.  Придумайте название (например, `balthomes-site`) и нажмите **Create repository**.
4.  Найдите ссылку **uploading an existing file** (текст "Get started by creating a new file or uploading an existing file"). Нажмите на неё.
5.  Откройте папку с этим файлом на своем компьютере. Выделите **ВСЕ** файлы и папки (`index.html`, `js`, `images`, `README.md`) и перетащите их в окно браузера на GitHub.
6.  Подождите, пока загрузятся файлы, и нажмите зеленую кнопку **Commit changes** внизу.
7.  **Как включить сайт**:
    - В репозитории нажмите **Settings** (шестеренка сверху).
    - В меню слева выберите **Pages**.
    - В разделе "Build and deployment" -> "Branch" выберите **main** (или master) и папку **/(root)**.
    - Нажмите **Save**.
    - Подождите 1-2 минуты, обновите страницу настройки Pages. Там появится ссылка на ваш сайт (например, `https://ващ-ник.github.io/balthomes-site/`).

## 2. Как настроить заявки в Telegram
Чтобы заявки с сайта приходили вам в личку:

1.  Создайте бота в Telegram:
    - Найдите бота **@BotFather**, нажмите Start.
    - Напишите `/newbot`.
    - Придумайте имя и юзернейм (должен заканчиваться на `bot`).
    - Скопируйте полученный **Token** (длинный набор букв и цифр).
2.  Узнайте свой ID:
    - Найдите бота **@userinfobot**, нажмите Start.
    - Скопируйте **Id**.
3.  Вставьте данные в файл:
    - На GitHub откройте файл `js/telegram-sender.js`.
    - Нажмите иконку карандаша (Edit).
    - Замените `'YOUR_BOT_TOKEN_HERE'` на ваш токен.
    - Замените `'YOUR_CHAT_ID_HERE'` на ваш ID.
    - Нажмите **Commit changes**.
4.  **Важно**: Напишите любой текст своему новому боту, чтобы активировать общение.

## 3. Админка (Программа для редактирования)
Я сделал для вас программу `admin.py`, чтобы добавлять квартиры без кодинга.

### Как запустить:
1.  Вам нужен Python (обычно есть на Mac).
2.  Откройте эту папку в Терминале.
3.  Напишите команду: `python3 admin.py`
4.  Откроется окно.
    - Слева список квартир.
    - Справа поля для редактирования.
    - Кнопка **"Выбрать файл"** сама скопирует фото в папку `images`.
    - Кнопка **"ЗАПИСАТЬ НА САЙТ"** обновит файлы `js/properties.js` и `data.json`.
5.  После того как вы нажали "ЗАПИСАТЬ", файлы на компьютере обновились. Теперь загрузите их на GitHub (пункт 1).

## 4. Как добавлять квартиры вручную (если без программы)
Вам не нужно быть программистом, чтобы менять объявления.
1.  Откройте файл `js/properties.js` (на GitHub или на компьютере).
2.  Там лежит список квартир.
3.  Чтобы добавить новую, скопируйте блок `{ ... },` и вставьте его ниже.
4.  Поменяйте `price`, `title`, `description` и путь к картинке.

## 4. Как добавить фото
1.  Загрузите фото в папку `images` (или через "Upload files" на GitHub в паку images).
2.  В файле `js/properties.js` укажите путь: `image: 'images/ваше-фото.jpg'`.
